# 💡 Insurance Premium Category Predictor

A full-stack Machine Learning project that predicts insurance premium categories (Low / Medium / High) based on user inputs.  
This project demonstrates end-to-end ML deployment using FastAPI, Docker, AWS EC2, and Streamlit.

---

## 🚀 Live Architecture

User → Streamlit UI → FastAPI API → ML Model → Prediction

## 🧠 Features

- 🎯 Predict insurance premium category  
- 📊 Displays prediction confidence & probabilities  
- 🧮 BMI calculation for better insights  
- 🌐 Real-time API integration  
- 🎨 Interactive UI using Streamlit  
- 🐳 Dockerized backend for consistency  
- ☁️ Deployed on AWS EC2 (24/7 availability)  

---

## 🛠️ Tech Stack

- Frontend: Streamlit  
- Backend: FastAPI  
- ML Model: Scikit-learn  
- Deployment: AWS EC2  
- Containerization: Docker  
- Language: Python
